Wordle Project

Last year, I created a Wordle game for my programming class using Java and JavaFX. I also used an external program called SceneBuilder to create an FXML file. This file allows the user to effortlessly play the game, connecting the game mechanics code to a visually appealling interface. This project allowed me to futher explore design within Computer Science.
